# Roadmap

This roadmap outlines the planned evolution of Journal Intelligence.

The project follows an incremental development approach, prioritizing stability, transparency, and usability over rapid feature expansion.

---

# Version 0.1 — Minimum Viable Product (Current)

Objective:

Provide transparent journal recommendations based on manuscript title and abstract.

Planned features:

- Semantic journal search
- Transparent ranking
- Simple Mode
- Advanced Mode
- Scopus and SINTA filtering
- APC filtering
- Language filtering
- Open Access filtering
- Explainable recommendations
- Export (CSV, JSON, Markdown)

---

# Version 0.5

Objective:

Improve recommendation quality and user experience.

Possible additions:

- Additional journal metadata
- More filtering options
- Better semantic search
- Improved recommendation explanations
- Enhanced journal detail pages

---

# Version 1.0

Objective:

Establish Journal Intelligence as a reliable open-source journal recommendation platform.

Possible additions:

- Support for full manuscript analysis
- Improved ranking customization
- Expanded journal database
- Faster search performance
- Improved accessibility

---

# Future Versions

Future development may include:

- Community-contributed metadata
- Review timeline information
- Historical journal information
- Recommendation comparison tools
- Plugin architecture
- AI-assisted features (optional)

Future features will only be introduced when they align with the project's principles.

---

# Guiding Philosophy

Journal Intelligence grows by solving one problem well before solving the next.

New features should improve researcher experience without reducing transparency or increasing unnecessary complexity.
