# Sample Dataset

## Purpose

This dataset provides a representative collection of academic journals for MVP development and testing.

It is intentionally small and is **not** intended to be a complete journal database.

## Current Schema

| Column | Description |
|---------|-------------|
| journal_name | Journal title |
| publisher | Journal publisher |
| indexing | Indexing system (Scopus, SINTA, etc.) |
| journal_rank | Quartile or SINTA level |
| apc_currency | APC currency |
| apc_amount | APC amount |
| language | Primary publication language |
| scope | Journal aims and scope |
| keywords | Keywords used for search and matching |
| submission_url | Official submission page |

Future versions may include additional metadata.
