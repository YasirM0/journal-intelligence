# Journal Intelligence — Vision

## Mission

Journal Intelligence is an open-source platform that helps researchers identify suitable academic journals using transparent semantic search and practical publishing information.

The project is designed to support researchers in making informed submission decisions without relying on opaque recommendation systems or paid AI services.

We believe journal recommendations should be transparent, inspectable, and grounded in verifiable evidence.

## Principles

### Transparency

Every recommendation should be explainable. Users should be able to understand why a journal was recommended and which factors influenced the ranking.

### Open Source

The project is developed in the open. Anyone can inspect the code, suggest improvements, report issues, or contribute new features.

### Modularity

The system is built from independent components so that data collection, ranking, search, and user interface can evolve separately.

### Evidence First

Recommendations should rely on verifiable information whenever possible. If information is unavailable or uncertain, the system should clearly communicate that rather than guessing.

### User Control

Users decide how much information to provide and whether optional features (such as future AI integrations or saved projects) are enabled. No personal data is stored without explicit consent.

### Accessibility

The software should be easy enough for first-time researchers while remaining flexible for experienced users.

## Version 0.1 Goal

Version 0.1 answers one question exceptionally well:

"Given my title and abstract, which journals are the best candidates for submission?"

Version 0.1 intentionally focuses on a small number of reliable features rather than a large number of experimental ones.

Included features:

* Title and abstract input
* Optional keywords
* Simple and Advanced modes
* Semantic journal matching
* Transparent ranking
* Journal filtering (Scopus, SINTA, APC, quartile, language where available)
* Explainable recommendations
* Export of search results

Deferred to future versions:

* Full manuscript analysis
* AI-assisted writing suggestions
* Submission history
* Community-contributed statistics
* Historical journal trajectory
* Acceptance probability estimation

## Long-Term Vision

Journal Intelligence aims to become a trusted open-source reference tool for journal selection by prioritizing transparency, reproducibility, and practical usefulness over complexity.
